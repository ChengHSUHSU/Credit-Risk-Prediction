# Credit-Risk-Prediction
Implement some binary classfication model and some data analysis on Germany Credit Risk Data.

## Requirements

  * Python 3.6
  * Sklearn
  * Pandas
  * xgboost
  * lightgbm
  * matplotlib
 
 ## Usage

To reproduce the experiments mentioned in the paper you can run the following command:

```bash
python main.py
```
